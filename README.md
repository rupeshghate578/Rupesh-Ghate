# Minimalist To-Do List App

A clean, functional, and user-friendly to-do list web application built with HTML, CSS, and vanilla JavaScript.

## 🎯 Features Implemented

### Core Features
- ✅ **Add Tasks**: Input field with priority selector (Low, Medium, High)
- ✅ **Mark Complete**: Checkbox to mark tasks as completed
- ✅ **Delete Tasks**: Remove individual tasks
- ✅ **Dynamic Rendering**: Tasks update in real-time
- ✅ **Task Styling**: Completed tasks appear strikethrough and faded
- ✅ **Task Ordering**: Pending tasks appear first, completed tasks at the bottom
- ✅ **Enter Key Support**: Add tasks by pressing Enter

### Additional Features
- ✅ **Filter Buttons**: All, Completed, Pending filters
- ✅ **Edit Functionality**: Edit existing tasks
- ✅ **Task Statistics**: Display count of pending and completed tasks
- ✅ **Priority Badges**: Visual indicators for task priority levels
- ✅ **Local Storage**: Tasks persist across page reloads
- ✅ **Smooth Animations**: Fade-in effects and transitions
- ✅ **Responsive Design**: Works on mobile, tablet, and desktop
- ✅ **Empty State**: Shows helpful message when no tasks exist
